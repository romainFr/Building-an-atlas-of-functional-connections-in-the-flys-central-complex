Drivers and neuron types used.  "Wolff Type Name" refers to the type names as described in \cite{Wolff2015}, "New Type Name" to the nomenclature for short names adopted in this paper (following \citealt{Turner-Evans2017a,Green2017}). Type description is the long name, following the guidelines of \cite{Wolff2015}. Pre and post regions are labeled based on anatomical characteristics. The fine subdivisions have been used to establish if two neurons are anatomically overlapping.
